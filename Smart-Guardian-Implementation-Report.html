تقرير بالميزات المنفذة والمعتمدة فقط — الإصدار الأول التطويري

ملخص المنتج

تم تطوير واجهة ويب عربية متجاوبة باسم Smart Guardian — الحارس الذكي، تعمل من الرابط نفسه على الكمبيوتر والجوال والأجهزة اللوحية. تشمل هوية بصرية خاصة، وحقوق الاستخدام باسم sultan alharbi.

الواجهة وتجربة الاستخدام

  • واجهة عربية متجاوبة تتكيف مع الجوال والكمبيوتر.
  • شعار خاص بالحارس الذكي باللغتين العربية والإنجليزية مع الحقوق.
  • بطاقة علوية موحدة تعرض الهوية ومعلومة الصلاة القادمة.
  • عبارات تحفيزية، نصائح للدراسة والعمل، وآيات قرآنية وأدعية متغيرة.
  • الآيات تظهر بوضوح مع اسم السورة ورقم الآية، دون ترجمة مرفقة.
  • العبارات الإنجليزية تظهر معها ترجمة عربية واضحة.
  • أسهم للتنقل يدويًا بين العبارات.
  • حقوق ثابتة في أسفل الواجهة، مع دعاء متغير لصاحب الفكرة والتصميم.
  • آلية تحديث للرابط نفسه؛ المستخدم يحدّث الصفحة ليصل للإصدار الجديد.

متابعة الاسم والتنبيه

  • خانة لاسم الطالب أو بدائل النطق، وتدعم كتابة أكثر من بديل بفواصل.
  • متابعة الاسم عبر ميكروفون الجهاز بعد ضغط المستخدم على زر البدء.
  • دعم عملي للاسم العربي والإنجليزي وبدائل النطق المسجلة.
  • زر مباشر لبدء وإيقاف متابعة الاسم.
  • عرض آخر نص تم سماعه لتسهيل معرفة حالة الاستماع.
  • تنبيه صوتي عند التعرّف على الاسم، مع منع تكرار التنبيه بشكل مزعج.
  • تنبيه مرئي أعلى الصفحة عند سماع الاسم.
  • جدولة اختيارية لبدء المتابعة وإيقافها بوقت محدد، مع خيار التكرار اليومي.

ربط Telegram للتنبيه عن بُعد

  • قسم مستقل لربط Bot Token وChat ID الخاصين بالمستخدم.
  • زر حفظ، وزر اختبار، وزر مسح الربط.
  • رسالة تنبيه مخصصة عند سماع الاسم.
  • شرح قابل للفتح والإغلاق لطريقة إنشاء البوت والحصول على البيانات اللازمة.
  • بيانات الربط تحفظ محليًا على جهاز المستخدم، ولا تُرسل إلى خادم خاص بالأداة.

مواقيت الصلاة

  • عرض الصلاة القادمة ومدة الوقت المتبقي لها داخل البطاقة العلوية.
  • إعدادات المدينة والدولة عند الحاجة.
  • زر لاستخدام الموقع الحالي مباشرة على الجوال أو الكمبيوتر بعد منح إذن المتصفح.
  • تحديث المواقيت وفق المدينة أو الإحداثيات المختارة.
  • تفعيل اختياري لتنبيه وقت الصلاة أثناء فتح الصفحة فقط.

الملاحظات السريعة

  • إضافة ملاحظة سريعة وحفظها محليًا للمستخدم.
  • عرض الملاحظات المحفوظة داخل الواجهة بعد الحفظ مباشرة.
  • تمييز الملاحظة بحسب وضعها: مستقل أو مجموعة.
  • تصدير الملاحظات بصيغة TXT.
  • خيار حفظ الملاحظات بصيغة PDF عبر نافذة الطباعة.

وضع المجموعة

  • وضع مستقل للمستخدم، ووضع مجموعة اختياري.
  • إنشاء مجموعة جديدة برمز مشاركة، أو الانضمام لمجموعة موجودة باستخدام الرمز.
  • نسخ رمز المجموعة لمشاركته مع الزملاء.
  • العودة للوضع المستقل في أي وقت.
  • اسم مستعار للمستخدم داخل المجموعة.
  • مشاركة الملاحظات مع المجموعة عند اختيار المستخدم لذلك.
  • ربط تقني بقاعدة بيانات Firebase مع دخول مجهول للمشاركين، دون صفحة تسجيل دخول للمستخدم النهائي.
  • شرح قابل للفتح والإغلاق لطريقة إنشاء المجموعة والانضمام إليها.

اقتراحات المستخدمين

  • قسم مستقل لإرسال اقتراح أو فكرة لتطوير الأداة.
  • إرسال الاقتراحات إلى مساحة الإدارة المخصصة للمشروع.
